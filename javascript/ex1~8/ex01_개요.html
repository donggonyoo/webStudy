자바스크립트 개요

html 페이지에 동작을 사용자의 요구에 맞도록 수행 기능
브라우저의 언어임
인터프리터 방식의 언어

//한줄주석 /*여러줄 주석(블럭주석)*/ //internal방식 console.log("hello world"); /*콘솔에 표시*/

자바스크립트의 특징

  • 접근제한자 개념X
  • 변수 선언이 필요없음 , 초기화시 자료형 결정
  • 웹 환경에 특화된 언어 ==> 보안 상의 이유로 기능제한됨
  • 단점 : 보안에 취약함{소스가 외부 공개됨}

사용법

  • 인라인 : 태그 내에 간단히 소스코드추가
  • 내부(internal) : html파일 내에 script태그로 소스코드 추가
  • 외부(external) : 별도의 .js파일로 소스코드추가 (css가져오는 느낌과비슷함)

<a href = 'javascript: 자바스크립트소스코드'> 내용</a>
